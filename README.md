# Blog app

### Setup (MacOS)

- Create a virtual environment in the root of the project:
  ```
  python3 -m venv .env
  ```
- Activate it using command `source`:
  ```
  source .env/bin/activate
  ```
- Install required python packages:
  ```
  pip install -r requirements.txt
  ```
- Run the server:
  ```
  python manage.py runserver
  ```
